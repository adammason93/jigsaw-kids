Optional full-canvas background art (Garden run)

File names (same folder as this file):
  field.png, woods.png, beach.png, snow.png, park.png, pool.png

Size: 900 x 400 pixels (matches the canvas). Safe zone: keep the "floor" for the runner
roughly level — the character’s feet sit at y = 360 (from the top).

If a PNG is present and loads, the game uses it for the whole scene. If a file is missing
or failed to load, the built-in drawn background is used instead.

You can make these in any image tool, or generate them elsewhere and drop them in here.
